# Лекция 2
## Задание

Приложение и функция, которые собирают основные новости с сайта lenta.ru. 
Структура данных структиурирована словарем и содержит:
* название источника
* наименование новости
* ссылку на новость
* дата публикации

  > Для парсинга использован XPath.
